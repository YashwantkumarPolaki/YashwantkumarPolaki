## 🚀 Flagship Project — NeuraMorphix

**AI for biological pattern similarity & early health-risk awareness**

Real-world problem: most people only find out about metabolic/health risk patterns after symptoms show up. NeuraMorphix analyzes biological/health data in real time to surface similarity patterns and risk signals early — explainably, not as a black box.

**What's built:**
- 🧬 Cosine similarity engine for biological pattern matching
- 🔍 Autoencoder for anomaly/pattern detection
- 📊 SHAP-based explainability layer (no black-box outputs)
- 🗣️ Gemini-powered narrative layer (explains results in plain language)
- 🗄️ Real-time architecture on Weaviate (vector DB)

**Tech Stack:**
```
ML/AI:      PyTorch, SHAP, Cosine Similarity, Autoencoders, Gemini API
Backend:    FastAPI, Weaviate
Frontend:   React
Data:       NHANES (real-world health dataset integration in progress)
```

**Status:** Core AI engine functional on synthetic data → integrating real NHANES data next | Team of 6 (UI/UX, Full Stack, IoT, Research, Mobile)
